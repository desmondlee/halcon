    方案设计 #

    系统精度 #

    做测量项目时，系统精度一般是公差带的0.1 ~ 0.2

    做引导贴合项目时，系统精度一般是公差带的0.2 ~ 0.3

    公差带 #

    公差带 = 上公差 - 下公差

    软件重复性经验值 #

    2 ~ 4 个像素点

    测量项目-计算像素当量 #

    系统精度 = 软件重复性经验值 * 像素当量

    其中，系统精度 = 公差带 * [0.1 ~ 0.2]，公差带值由客户提供

    例，测量硬币直径时，若公差为±0.1mm(即公差带为0.2)，系统精度相对于公差带的系数取 0.2，软件重复性值为4个像素点，则像素当量 = ((0.1 + 0.1) * 0.2) / 4= 0.01 ，单位 mm/pixel。

    检测项目-计算像素当量 #

    • 一维码：基于灰度投影，最小条纹 ≥ 1.5 pixel
    • 二维码：基本单元用 3 ~ 6 个pixel以上表征
    • OCR/OCV：最窄笔画≥ 3pixel，建议 5 ~ 7pixel以上
    • 缺陷/划痕检测：最窄 ≥ 5pixel
    • 色差：对比度＞ 20灰度值

    选镜头 #

    远心镜头 #

    1. 放大倍率
    2. 兼容靶面尺寸
    3. 工作距离
    4. 景深

    注，放大倍率 = 传感器尺寸 / 视野

    FA镜头 #

    1. 放大倍率
    2. 工作距离
    3. 焦距 = 放大倍率 * 工作距离，根据焦距差镜头手册选镜头

    选相机 #

    多相机拼接 #

    1. 根据视野大小、像素当量，计算出相机分辨率，除以单个相机的分辨率，向上取整，即可得到相机个数；
    2. 多个相机并排放置拍照（相邻相机的视野会有重叠）；
    3. 使用一个大于视野范围的标定板（棋盘格、圆点类型等）（每个相机要保证能拍到标定板上最少9个特征点，否则精度不高）；
