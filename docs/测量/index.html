    测量 #

    工业测量的基础——卡尺工具。

    卡尺工具过程 #

    1. 确定边缘检测方向；
    2. 高斯滤波；
    3. 执行投影处理；
    4. 执行微分（对比度）处理；
    5. 设置边缘敏感度（对比度阈值）、边缘极性、边缘的位置；
    6. 边缘执行亚像素处理。

    一维测量流程 #

    graph TD A["采集图像"] --> B["创建测量对象" ] B --> C["测量"] C --> D["销毁测量对象"]

    需要熟练使用的一维测量工具：测量助手

    二维测量流程 #

    graph TD A["采集图像"] --> B["按顺序创建边缘测量ROI(卡尺工具)" ] B --> C["边缘测量"] C --> D["把边缘连接成XLD多边形轮廓"] D --> E["拟合"]

    拟合方法：

    1. 回归线法（无损拟合）
    2. 加权法（奇异点进行加权处理，距离XLD越远，权重越小）：huber
    3. 抛点法（抛除奇异点）：tukey
